Definicje w trójkącie prostokątnym

Dla kąta \(\alpha\):

\[ \sin\alpha = \frac{\text{przeciwległa}}{\text{przeciwprostokątna}},\quad \cos\alpha = \frac{\text{przyległa}}{\text{przeciwprostokątna}},\quad \tan\alpha = \frac{\sin\alpha}{\cos\alpha}. \]

Najważniejsze własności

\[ \sin(x+2\pi)=\sin x,\quad \cos(x+2\pi)=\cos x \] \[ -1 \le \sin x \le 1,\quad -1 \le \cos x \le 1 \] \[ \sin(-x)=-\sin x,\quad \cos(-x)=\cos x \]

Często używana tożsamość: \(\sin^2 x + \cos^2 x = 1\).

Szybkie zadanie

Oblicz: \(\sin 30^\circ\), \(\cos 60^\circ\), \(\tan 45^\circ\).

Pokaż odpowiedź
\[ \sin 30^\circ=\frac12,\quad \cos 60^\circ=\frac12,\quad \tan 45^\circ=1 \]

Co znajdziesz dalej?

  • Okrąg jednostkowy: \(P(\cos\alpha,\ \sin\alpha)\)
  • Wykresy: amplituda i okres
  • Tożsamości: suma i różnica, podwójny kąt
  • Zastosowania: pomiary i fale
